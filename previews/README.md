# Template Preview Images

This directory contains preview images (PNG/JPG) for each resume template.

## Required Files

Each template defined in `src/config/templates.ts` requires a corresponding preview image:

- `modern.png` - Preview for Modern template
- `template-2.png` - Preview for Classic template
- `template-3.png` - Preview for Executive template
- `template-4.png` - Preview for Creative template
- `template-5.png` - Preview for Minimalist template
- `template-6.png` - Preview for Complete template
- `template-7.png` - Preview for Trendy template
- `template-8.png` - Preview for AST template
- `template-9.png` - Preview for Professional template
- `complete.png` - Preview for Complete template

## Image Specifications

- **Format**: PNG or JPG
- **Recommended Size**: 800x1000px (or similar aspect ratio)
- **Content**: Should show a sample resume rendered with the template
- **Purpose**: Used in frontend template selection UI

## Generating Preview Images

Preview images can be generated by:

1. Rendering each template with sample data
2. Taking a screenshot or exporting as image
3. Saving to this directory with the correct filename

The preview images are served statically at `/previews/{filename}` by the backend.
